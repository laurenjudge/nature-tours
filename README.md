# Natures Project

This project uses CSS/Sass animations.
Created for a fake company that advertises their nature tours.

## Getting Started

Install [Node.js](https://nodejs.org/en/)

In commandline type:

$ npm install npm@latest -g
$ npm install

Open index.html
